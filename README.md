# TINOS CUP — Football Tournament Site

Στατικό site (HTML/CSS/JS, χωρίς build) για τα τουρνουά Tinos Cup:
αποτελέσματα, **αυτόματη βαθμολογία** (Νίκη 3 / Ισοπαλία 1 / Ήττα 0),
εναλλαγή **περιόδων**, **tabs** ανά ηλικιακή κατηγορία και **overview**.

## Άνοιγμα τοπικά
Διπλό κλικ στο `index.html`, ή για να δουλέψουν σωστά τα σχετικά αρχεία:
```bash
npx http-server -p 8099      # και άνοιξε http://localhost:8099
```

## Σελίδες
- `index.html` — αρχική: επιλογή **ΑΓΟΡΙΑ** ή **ΚΟΡΙΤΣΙΑ**.
- `boys.html` — δημόσια σελίδα για τα αγόρια (όλες οι περίοδοι/ηλικίες αγοριών).
- `girls.html` — δημόσια σελίδα για τα κορίτσια.
- `admin.html` — διαχείριση σκορ **όλων** (αγόρια + κορίτσια μαζί), μόνο για σένα.

Αγόρια/κορίτσια είναι ξεχωριστές σελίδες ώστε η περίοδος που βλέπεις να μη
"χάνεται" όταν αλλάζεις κατηγορία. Το φύλο της σελίδας ορίζεται με
`window.TINOS_GENDER` πριν φορτώσει το `js/app.js`.

## Πώς ενημερώνω (admin)
Στο `admin.html` επεξεργάζεσαι **τα πάντα**: ομάδες, αγωνιστικές, ποιος παίζει
με ποιον, ημέρες/ώρες, σκορ, ρεπό, τρέχουσα περίοδος. Η βαθμολογία ενημερώνεται
**ζωντανά** (προεπισκόπηση).

**Αυτόματη δημοσίευση (κουμπί «Αποθήκευση & Δημοσίευση»):**
1. Μία φορά: πάτα **⚙ Ρυθμίσεις** και βάλε ένα GitHub **fine-grained token** με
   δικαίωμα **Contents: Read and write** μόνο στο repo `dimliaro/tinos-cup`.
   (Αποθηκεύεται μόνο στον browser σου.)
2. Κάνε τις αλλαγές σου και πάτα **💾 Αποθήκευση & Δημοσίευση**. Γίνεται commit
   το `js/data.js` στο GitHub και το live site ενημερώνεται σε ~1 λεπτό.

Εναλλακτικά (χωρίς token): **⤓ Export** κατεβάζει `data.js` για χειροκίνητο commit.

## Πού αλλάζω ομάδες / αγώνες / περιόδους
Όλα στο `js/data.js` — έχει σχόλια. Δομή:
`tournament → periods[] → categories[] → teams[] / fixtures[] → matches[]`.
Λογότυπα ομάδων: βάλε `logo: "assets/logos/ομαδα.png"` σε κάθε ομάδα
(αλλιώς δείχνει αρχικά με χρώμα).

## Ανέβασμα στο διαδίκτυο (GitHub Pages, δωρεάν)
1. Φτιάξε public repo `tinos-cup` στο GitHub.
2. Ανέβασε όλα τα αρχεία.
3. Settings → Pages → Branch `main` / `/root` → Save.
4. Σε ~1 λεπτό: `https://<username>.github.io/tinos-cup`.
